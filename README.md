## 📖 About

**JSON Path Finder** is online tool and Chrome Extension that helps developers navigate complex JSON documents with ease.

Instead of manually tracing nested objects and arrays, simply click on any value in the JSON tree and instantly get its path in your preferred format.

Perfect for debugging API responses, working with webhooks, creating automations, or exploring configuration files, JSON Path Finder makes working with JSON significantly faster.

Everything runs locally in your browser, so your data never leaves your computer.
